# Talent Bridge Console

## Project Overview
Talent Bridge is a recruitment platform designed to connect job seekers with recruiters efficiently. The goal of the project is to create a full-stack application where companies can post job openings and candidates can apply for them.

This project is currently in the **initial development phase**, where the basic project structure and version control setup have been established.

---

## Current Project Structure

Talent_Bridge_console

client/ → Frontend application  
server/ → Backend application  
.gitignore → Git ignored files  
README.md → Project documentation

## Work Completed So Far

### 1. Project Initialization
The project was initialized locally and prepared for development.

Key actions:
- Created the main project folder
- Organized the application into frontend and backend directories.

---

### 2. Full-Stack Project Architecture
The project follows a **full-stack architecture** with separate folders for frontend and backend development.

client/ → Frontend (React – planned)  
server/ → Backend (Node.js + Express – planned)

This structure allows independent development of UI and server logic.

---
